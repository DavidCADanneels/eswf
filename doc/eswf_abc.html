Description

The eswf_abc module provides functions for manipulating ActionScript Byte Code segments (herein ABC segments). For details on the ABC data format, see chapter 4 of ActionScript Virtual Machine 2 Overview.

stringmap/2

stringmap(Fun, ABCSegment::binary()) -> iodata()

  • Fun = (String::binary()) -> iodata() | skip

Returns an ABC segment where each string constant String in ABCSegment has been replaced by Fun(String). Fun(String) must return a valid UTF-8 sequence. Alternatively, Fun(String) may return skip, and String will remain unmodified in the resulting ABC segment. (Returning skip is semantically equivalent to returning String but slightly faster.)

Except as mentioned above, the returned ABC segment does not otherwise differ from ABCSegment. The result of applying stringmap to an invalid ABC segment is undefined.
