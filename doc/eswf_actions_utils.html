stencilify/3

stencilify(Actions::binary(), Fun, UserAcc::Acc) -> {Chunks, NewAcc}

  • Fun = (String::binary(), Acc) -> {Action, NewAcc}
  • Action = skip | {punch, Key}
  • Chunks = [{chunk, iodata()} | {hole, Key}]
  • Key = term() | {as2insnlen, BaseSize, [Key]}

Returns a list of chunks after folding Fun over each string in a sequence of ActionScript instructions. To generate a valid sequence of ActionScript instructions, each as2insnlen hole must be filled with the 16-bit representation of BaseSize plus the size of each value specified in Key. Every other hole should be replaced by a null terminated ASCII string.
