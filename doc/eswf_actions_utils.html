Function Details

stringmap/2

stringmap(Fun, Actions::iodata()) -> iodata()

  • Fun = (String::binary()) -> iodata() | skip

Returns a sequence of ActionScript (1.0 or 2.0) instructions where each dictionary string String in Actions has been replaced by Fun(String). Fun(String) must not contain any null bytes. Alternatively, Fun(String) may return skip, and String will remain unmodified. (Returning skip is semantically equivalent to returning String but slightly faster.)

Except as mentioned above, the returned ActionScript instruction sequence does not otherwise differ from Actions. The result of applying stringmap to an invalid instruction sequence is undefined.
